# 🌙 Ramadan Python Challenge 2026

A 30-day Python coding challenge designed for DevOps engineers who want to sharpen their programming fundamentals while observing Ramadan.

## About

One problem a day, every day of Ramadan. Each challenge blends classic data structure and algorithm problems with practical DevOps scenarios — log parsing, config management, networking, CI/CD patterns, and more.

## Structure

- **Week 1:** Warm-Up & Foundations
- **Week 2:** Strings, Files & Automation
- **Week 3:** Data Structures & DevOps Patterns
- **Week 4:** Graphs, Trees & Real-World DevOps
- **Final Days:** Capstone Challenges
- **Bonus:** 5 extra challenges if you finish early

Each day's solution lives in its own file: `day01_reverse_log.py`, `day02_count_unique_ips.py`, etc.

## Code Style

- Python 3.10+
- Type hints on all function signatures
- Google-style docstrings
- `if __name__ == "__main__":` block with test cases
